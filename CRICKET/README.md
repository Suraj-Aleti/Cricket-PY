# ?? CricketPY: AI-Powered Chase Predictor

An AI-powered T20 match simulator that predicts the outcome of a second-innings run chase using Machine Learning.

## ?? Key Features
* **Hybrid Prediction Engine:** Combines classification and regression models.
* **Environmental Logic:** Adjusts for Pitch Compaction, Grass, and Dew Factor.
* **Simulated Target Path:** Calculates the required runs for each remaining over.

## ??? Setup
1. Clone the repo: \git clone https://github.com/Suraj-Aleti/Cricket-PY.git\
2. Install libraries: \pip install -r requirements.txt\
3. Run: \py ai_predictor.py\
